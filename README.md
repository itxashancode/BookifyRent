## 📌 About The Project

**BookifyRent** is a modern, responsive **book rental web platform** that gives readers access to thousands of textbooks and bestsellers — for a fraction of the purchase price. Whether you're cramming for finals, diving into fiction, or leveling up your business skills, BookifyRent puts your next read just one click away.

> *"Your digital library is just a click away."*

<br/>

## ✨ Features

| Feature | Description |
|---|---|
| 🔍 **Smart Search** | Search across thousands of books from the navbar instantly |
| 📚 **Featured Books** | Curated *"Featured & New Drops"* section with weekly pricing |
| 🗂️ **Category Filter** | Scroll filter for Finals Prep, Fiction, STEM, Business, Self-Help & more |
| 🛒 **Rent Now Modal** | Preview book details, choose rental duration (1 week / 2 weeks / 1 month) |
| 💡 **How It Works** | 3-step visual guide — Search → Rent → Read |
| 💰 **Pricing Plans** | Free, Premium ($9.99/mo), and Student ($4.99/mo) tiers |
| 📱 **Fully Responsive** | Mobile-friendly with a sticky Rent Now CTA on smaller screens |
| 🎞️ **AOS Animations** | Smooth scroll-triggered fade, zoom, and flip animations throughout |
| 🌐 **Social Links** | Facebook, Twitter, Instagram & TikTok in footer |
| 📲 **App Download CTAs** | App Store & Google Play buttons ready in footer |

## 🚀 Getting Started

### Prerequisites

No build tools needed — just a modern browser.

### Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/itxashancode/BookifyRent.git

# 2. Navigate into the project
cd BookifyRent

# 3. Open in your browser
open index.html
```

> 💡 For the best dev experience, use **Live Server** in VS Code (`Right-click index.html → Open with Live Server`)
